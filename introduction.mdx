<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-2xl sm:text-3xl font-semibold text-gray-900 dark:text-white mb-4">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get up and running with GitSpace in minutes</p>
  
  <Steps>
    <Step title="Install GitSpace">
      Install the `gssh` CLI tool using your preferred package manager:
      
      <CodeGroup>
      ```bash npm
      npm install -g gitspace
      ```
      ```bash bun
      bun install -g gitspace
      ```
      ```bash pnpm
      pnpm install -g gitspace
      ```
      ```bash yarn
      yarn global add gitspace
      ```
      </CodeGroup>
      
      <Note>
        GitSpace requires [GitHub CLI](https://cli.github.com/) (`gh`), [Git](https://git-scm.com/), and [jq](https://stedolan.github.io/jq/) to be installed. Authenticate with `gh auth login` before using GitSpace.
      </Note>
    </Step>
    
    <Step title="Launch the TUI">
      Start the interactive terminal user interface:
      
      ```bash
      gssh
      ```
      
      The TUI provides a two-panel interface for managing projects and workspaces. Use **Tab** to switch panels, **n** to create new items, and **?** for help.
    </Step>
    
    <Step title="Add your first project">
      Add a GitHub repository as a project:
      
      ```bash
      gssh project add
      ```
      
      GitSpace will clone the repository to `~/gitspace/<project-name>/base` and detect the default branch automatically.
    </Step>
    
    <Step title="Create a workspace">
      Create an isolated workspace for feature development:
      
      ```bash
      gssh workspace add feature-name --project my-project
      ```
      
      This creates a new git worktree at `~/gitspace/<project-name>/workspaces/feature-name/` with its own branch. Work on multiple features simultaneously without stashing changes.
    </Step>
  </Steps>
</div>
